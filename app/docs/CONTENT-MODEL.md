# Conteúdo editável

O painel permite editar sem alterar HTML, CSS ou TypeScript:

- logo e favicon;
- nome, título profissional, telefone, WhatsApp, endereço e redes sociais;
- SEO e texto do rodapé;
- primeira tela e imagens principais;
- títulos, textos, ordem e visibilidade das seções;
- procedimentos;
- antes e depois com autorização;
- depoimentos com autorização;
- perguntas frequentes;
- biblioteca de imagens;
- Meta Pixel, Meta CAPI, GA4 e Google Ads.

O layout continua protegido pelo código. O administrador altera conteúdo, não seletores CSS ou scripts.
